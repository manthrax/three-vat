This is a codebashed implementation of Houdini Vertex Animation Textures (VAT3 and VAT2) for THREEJS.
It uses shader injection to allow use of other threejs material types, and supports instancing with per instance start time.
It was cobbled together using these repos/resources as a reference:

https://github.com/mikelyndon/r3f-webgl-vertex-animation-textures
https://github.com/floating-world-lda/vat3-wgsl-ts/tree/main/three-js
https://github.com/floating-world-lda/vat3-wgsl-ts/tree/main/babylon-js
